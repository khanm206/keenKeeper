# 🚀 KeenKeeper — Keep Your Friendships Alive

> A modern friendship tracking web app that helps you stay connected with the people who matter.

---

## 🌐 Live Demo

🔗 https://keen-keeper-mk.netlify.app

---

## ✨ Features

### 🏠 Home Dashboard

- View all your friends in a clean card layout
- Status-based highlights (Overdue / Almost Due / On-Track)
- Quick overview of your relationships

---

### 👤 Friend Details Page

- Detailed friend profile
- Relationship goal tracking
- Quick actions:
  - 📞 Call
  - 💬 Text
  - 🎥 Video Chat
- Automatically logs interactions into timeline

---

### 📜 Timeline System

- Track all your interactions in one place
- Displays:
  - Date
  - Action type (Call / Text / Video)
  - Friend name
- Filter interactions by type

---

### 📊 Analytics (Stats Page)

- Visual representation using Pie Chart
- Shows interaction distribution:
  - Calls
  - Texts
  - Video chats

---

### 🎯 Smart Tracking

- Tracks:
  - Days since last contact
  - Next due date
  - Relationship goals

---

### 📱 Fully Responsive

- Works on:
  - Mobile 📱
  - Tablet 📟
  - Desktop 💻

---

## 🛠️ Tech Stack

- ⚛️ React.js
- 🎨 Tailwind CSS + DaisyUI
- 🔀 React Router DOM
- 📊 Recharts
- 🌐 Axios (for API/data handling)
- 🔔 React Toastify (for notifications)

---

## 🔔 Notifications

This project uses **React Toastify** to show real-time user feedback:

- Call/Text/Video actions trigger success notifications
- Improves user experience with instant feedback

---

## 🌐 API / Data Handling

This project uses **Axios** for:

- Fetching friends data
- Handling asynchronous requests cleanly
- Managing API calls in a structured way

---

## 🧠 Key Learnings

- State management using Context API
- Dynamic routing with React Router
- Data visualization using Recharts
- Responsive UI design with Tailwind
- Real-world UI/UX structuring
